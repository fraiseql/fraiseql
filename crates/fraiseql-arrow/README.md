# fraiseql-arrow

Apache Arrow Flight integration for FraiseQL. This crate bridges the FraiseQL query engine with the Arrow ecosystem, enabling high-throughput columnar data exchange for analytical workloads over the Flight SQL protocol.

## Features

- Arrow Flight SQL streaming
- Columnar data exchange
- Zero-copy data transfer
- Integration with Arrow ecosystem
- High-throughput analytical queries

## Usage

```toml
[dependencies]
fraiseql-arrow = "2.1.0"
```
