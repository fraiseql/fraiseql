# Tutorials

Learn FraiseQL by building real-world applications step-by-step.

## [Building a Blog API](./blog-api.md)

Create a complete blog API with users, posts, comments, and authentication. This tutorial covers:

- Setting up PostgreSQL tables and views
- Defining GraphQL types with relationships
- Implementing queries and mutations
- Adding authentication and authorization
- Testing your API

Perfect for beginners to get hands-on experience with FraiseQL's core concepts.

## Prerequisites

All tutorials assume you have:
- Python 3.13+
- PostgreSQL 14+
- Basic knowledge of GraphQL
- Familiarity with SQL

Ready to start building? Begin with the [Blog API tutorial](./blog-api.md)!
