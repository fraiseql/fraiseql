# FraiseQL Migration Guides

This directory contains guides for migrating to FraiseQL from other GraphQL solutions.

## Available Guides

### [From Hasura/PostGraphile](./from-hasura-postgraphile.md)
Comprehensive guide for migrating from Hasura or PostGraphile to FraiseQL, including:
- Architecture comparison
- Schema migration patterns
- Permission system migration
- Mutation and subscription patterns
- Deployment strategies
- Performance optimization

## Why Migrate to FraiseQL?

### Simplicity
- Pure Python implementation
- No external runtime dependencies
- Standard FastAPI application

### Performance
- ~10x lower memory usage than Hasura
- Faster startup times
- Efficient query generation

### Developer Experience
- Full type safety with Python hints
- Native debugging tools
- Familiar Python ecosystem

### Flexibility
- Not tied to database schema
- Custom business logic in Python
- Easy integration with existing systems
