# .claude/ Directory

This directory contains configuration and context for AI-assisted development sessions.

## Committed Files

| File | Purpose |
|------|---------|
| `CLAUDE.md` | Project instructions for AI sessions (authoritative) |
| `IMPLEMENTATION_ROADMAP.md` | Feature implementation status (authoritative) |

## What Is NOT Committed Here

Session work plans, scratch notes, quality analysis artifacts, and documentation reviews
are gitignored (see `.claude/.gitignore`). They exist only in the local working directory
as AI session context.

## For Human Maintainers

You do not need to read this directory. The relevant documentation lives in:

- **Architecture**: [`docs/architecture/`](../docs/architecture/README.md)
- **Development workflow**: [`CONTRIBUTING.md`](../CONTRIBUTING.md)
- **Feature roadmap**: [`ROADMAP.md`](../ROADMAP.md)
- **Operational runbooks**: [`docs/runbooks/`](../docs/runbooks/)
