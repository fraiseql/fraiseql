╚════════════════════════════════════════════════════════════════════════════╝

✓ You can always ask the AI to explain something before correcting it
✓ The AI will track all changes and commit them automatically
✓ You're in control - go as fast or slow as you want
✓ You can come back later to resume the review
✓ Each file is reviewed completely before moving to the next
✓ The AI will show before/after of all corrections
✓ You can request additional changes to the same file
