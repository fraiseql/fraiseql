Before You Start

Alpha Software ⚠️ Please read carefully

⚠️ Alpha Software Notice

FraiseQL is currently in alpha stage (v0.1.0a10). This means:

  • APIs may change without notice
  • Features may be incomplete or buggy
  • Not suitable for production use
  • Limited documentation and examples
  • Security features are incomplete

We recommend using FraiseQL only for:

  • Learning and experimentation
  • Proof of concept projects
  • Contributing to development

Requirements

PostgreSQL 13+

FraiseQL requires PostgreSQL 13 or higher with JSONB support.

Other databases are not supported and there are no plans to add support.

Python 3.11+

Full type hints support requires Python 3.11 or higher.

Python 3.10 may work but is not officially supported.

Basic SQL Knowledge

You'll need to write PostgreSQL views and functions.

FraiseQL doesn't generate database schemas automatically.

What's NOT Ready

🚨 Security

  • No rate limiting
  • No CSRF protection
  • No security headers
  • Limited error handling

📊 Performance

  • No result caching
  • No query complexity limits
  • Limited connection pooling options
  • No production benchmarks

🔧 Operations

  • No database migrations
  • No backup strategies
  • Limited deployment docs
  • No monitoring dashboards

✨ Features

  • No GraphQL subscriptions
  • No real-time updates
  • No file uploads
  • No federation

If You Want to Continue...

1. Install Alpha Version

pip install fraiseql==0.1.0a10

Pin to specific alpha version to avoid breaking changes

2. Use Development Mode Only

app = create_fraiseql_app(
    database_url="postgresql://localhost/test",
    types=[User],
    production=False  # Never use True in alpha
)

3. Expect Breaking Changes

Follow the release notes carefully. APIs will change between alpha releases.

Production-Ready Alternatives

If you need a production GraphQL solution today, consider:

Hasura

Auto-generated GraphQL from database schema. Production ready with enterprise support.

PostGraphile

Instant GraphQL API from PostgreSQL schema. Mature and battle-tested.

Strawberry GraphQL

Python GraphQL library with great DX. What FraiseQL's API is inspired by.
